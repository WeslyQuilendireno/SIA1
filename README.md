# System Infrastructure and Architecture 1 — Machine Problems

This repository contains a collection of Python programs developed for the System Infrastructure and Architecture 1 course. The projects demonstrate core programming concepts, data structures, error handling, and file processing.

##  Tech Stack & Concepts Used
* **Language:** Python 3.x
* **Core Concepts:** Control structures, iteration, functions, and file I/O
* **Data Structures:** Lists, Dictionaries
* **Error Handling:** Try-Except blocks for robust execution

---

## Included Problems & Features

### 1. Student Grade Averager (Basic)
* Accepts student name and grades for Math, Science, and English.
* Calculates and displays the general average.

### 2. Mad Libs Song Lyric Replacer
* Accepts 3 nouns and 3 adjectives from the user.
* Replaces words in a favorite song lyric with the user inputs in **ALL CAPS**.
* Displays both the original and modified lyrics side-by-side.

### 3. Employee Bonus Calculator
* Determines employee bonus based on years of service and department initials (`IT`, `Acct`, `HR`).

### 4. Automated Payslip Generator
* Accepts employee details, hours worked, and deductions (SSS, PhilHealth, Housing Loan).
* Computes and renders a structured payroll payslip.

### 5. Iterative Student Grade Calculator
* Loops dynamically to process multiple student records using functions.
* Accepts 4 parameters: Name, Math, English, and Science grades.

### 6. String Reversal & Analyzer
* Reverses user-inputted strings and converts them to uppercase using a custom function.
* Displays original text, modified text, and total character count.

### 7. Word Bank Application
* Dynamically stores user-inputted words into a list structure.
* Features a retry loop (`Y/N`) and displays total word counts upon exit.

### 8. Dictionary-Based Record Keeping App
* Console app managing data storage using Python dictionaries.
* Supports dynamic key-value pairing for adding and deleting specific records.

### 9. File-Based Record Keeping System
* Persistent CRUD system storing data (Name, Email, Address) directly into a `.txt` file.
* Features options to Add, View All, and Clear records without deleting the underlying file.

### 10. Robust Calculator App
* Performs Addition, Subtraction, Multiplication, and Division on two numbers.
* Implements strict exception handling for `ZeroDivisionError` and `ValueError` (text inputs).

2. Navigate to the project directory:
   ```bash
   cd YOUR-REPO-NAME
   ```
3. Run any machine problem file using Python:
   ```bash
   python name_of_file.py
   ```
