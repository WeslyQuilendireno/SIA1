# System Infrastructure and Architecture 1 — Machine Problems

A collection of Python programs developed for the System Infrastructure and Architecture 1 course. Each file corresponds to a machine problem (MP) or laboratory activity (LA) covering fundamental programming concepts — from basic I/O and control flow to file handling and exception management.

##  Tech Stack & Concepts Used
* **Language:** Python 3.x
* **IDE:** PyCharm
* **Concepts Covered:** Control flow, iteration, functions, string manipulation, file I/O, data structures
 (lists & dictonaries), exception handling

## How to Run the Programs

1. Clone this repository:
   ```bash
   git clone https://github.com/WeslyQuilendireno/SIA1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd YOUR-REPO-NAME
   ```
3. Run any machine problem file using Python:
   ```bash
   python name_of_file.py
   ```
